Node.js е софтуерна система oт страна на сървара, проектирана за писнае на мащабни интернет приложения, oсобено за web serverи. Програмите се пишат от страна на сървъра с JavaScript, използващ event-driven, asynchronous I/O за да се минимизира оverhead-а и да се мakксимизира мащабността. 

Node.js създава web server от самосебе си, правейки ненужно използването на web server software, като Apache или Lighttpd. Node.js позволява пълен контрол върху това, как web servera работи. Node.js позволява web разработчиците да създават цялостни web приложения и в сървърната, и в клиентската част, използвайки само един език - JavaScript.

Details
------------------------------------------------------
Node.js e пакетирана компилация от Google's V8 JavaScript engine, libUV аbstraction layer платформа и core библютека, която само по себе си е написна на node.js.

Node.js е създаден от Ryan Dahl. Проекта е започнат от 2009 година и разработката и подръжката е спонсорирана от Joyent, нейния бивш работодател.

Първоначалната цел на Dahl е да създава web сайтове с възможности който се виждат в други web приложения като GMail. Пробвайки различни решения в няколко различни програмни езици, той избира JavaScript поради липсата на I/O API. Това му позволява да дефинира конвенция за non-blocking, event-driven I/O.

Подобни среди написани на различни езици включително Tornado и Twisted за Python, Perl Object Environment for Perl, libevent за C, Vert.x за Java, Akka за Java, Event Machine for Ruby. Различен от повечето JavaScript програми, не се изпълнява в браузъра, а от страна на сървъра. Node.js имплементира някой CommonJS спецификации. Той също представя и REPL среда за интерактивно тестване.

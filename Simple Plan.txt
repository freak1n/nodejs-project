1. Общо представяне за node.js
	- server-side софтуерна система предназначена за писане на мащабни интернет приложения, особено web сървари
	- програмите се пишат на сървара с помоща на JavaScript, използващ event-driven архитектура, асинхронен вход/изход за да минимизира overhead-a и да увеличи скалираемостта
	- създава web server от само себе си, така че прави използването на web server-и като Apache или Lighttpd и позволява пълен контрол върху работата и настройките на web server-a
	- позволява web разработчиците цяло web приложение със server-side и client-side само с единствен език - JavaScript

3. В детайли
	- node.js e пакетирана компилация от V8 JavaScript engine-a на Google, libUV платформа абстрактен слой и core библиотеката написана на чист JavaScript
	- създадена от Rayn Dahl starting in 2009 и до сега е разработвана и подържана от Joyent 
	- Първоначално идеята била да се използва за създаване на web сайтове за push съвместимост каквато се вижда и в web apps като Gmail. 
	- Като повечето js програми, node.js не се изпълнява от браузъра, а вместо това от server-side JavaScript app.
	- Имплементира някои CommonJS спецификации. Също така осигурява REPL среда за интерактивно тестване.
